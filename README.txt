How to use the program:

1. Once you run the program you will be brought to a Login screen. You can enter either "test" or "admin" as both the username and password.
2. When logged in, it will show you the appointment screen with a table of all appointments.
3. From there you can add, edit or delete an appointment. You can also click the Customers or Report button.
4. If you click the Customers button, you can add, edit or delete a customer.
5. If you click the Reports button, you can generate any of the 3 reports your interested to examine.


Additional report in part A3f: My additional report I chose to generate was a report that shows all appointments for a specific customer.
There is a drop down box in the report, from that you can choose any customer in the application to view all of their appointments.
